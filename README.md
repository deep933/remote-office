
## WebRTC Muti peer (Mesh) video conferencing app with signaling server

## Built with
* [React.js](https://reactjs.org/)
* [Socket.io](https://socket.io/)
* [Node (Express.js)](https://expressjs.com/)
* [Tailwind css](https://tailwindcss.com/)

## Getting Started
```sh
npm install 
```
```sh
npm start
```
